# 🥥 AI Nutrition Coach for Keralites

**🚀 Live Demo:** [Add your Streamlit URL here] 
**📂 GitHub:** https://github.com/Susmithasuraj/ai-nutrition-coach-kerala

**Zeroth Phase MVP** - Personalized 7-day Kerala meal plans for weight loss, postpartum, thyroid, PCOS

### ✨ Features
- **BMR/TDEE Calculator** - Mifflin-St Jeor formula based on age, weight, height, activity
- **7-Day Meal Planner** - Monday-Sunday customizable Kerala meals with dropdowns
- **Auto Calorie Tracking** - Daily total calculated vs target with alerts
- **Shopping List Generator** - Weekly grocery list in 1 click
- **Goal-based Logic** - Weight Loss (-400 kcal), Postpartum (+500 kcal), Muscle Gain (+300 kcal)
- **Health Alerts** - Thyroid warnings: avoid raw cabbage/cauliflower, PCOS-friendly options
- **100+ Verified Foods** - ICMR + Kerala Agri Uni database: Puttu, Appam, Meen Curry, Cheera Thoran

### 🛠️ Tech Stack
- **Frontend**: Streamlit
- **Backend**: Python, Pandas
- **Data**: CSV-based Kerala food database
- **Deployment**: Streamlit Cloud
