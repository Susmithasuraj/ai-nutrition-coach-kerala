# 🥥 AI Nutrition Coach for Keralites

**Zeroth Phase MVP** - Personalized Kerala meal plans for weight loss, postpartum, thyroid, PCOS

### ✨ Features
- BMR/TDEE Calculator based on age, weight, height
- Kerala-specific food database: Puttu, Appam, Meen Curry, Cheera Thoran
- Special modes: Postpartum +500kcal, Thyroid warnings, PCOS
- 100+ ICMR + Kerala Agri Uni verified foods

pip install -r requirements.txt
